🎓 Sessional Marks Calculator (Teacher-Focused System)

📌 Overview

This project is a Sessional Marks Calculator System designed specifically for teachers.
It helps teachers manage, calculate, and export student sessional results efficiently.

Instead of manual calculations, this system automates:

- Marks aggregation
- Weightage handling
- Grade calculation
- Class ranking
- PDF result generation


🚀 Features

- 🔐 Teacher Authentication
  
  - Create account / Login system
  - Secure access for teachers only

- 📊 Course & Metadata Setup
  
  - Department selection
  - Semester selection
  - Course-related details

- 🧮 Flexible Marks Input
  
  - Mid-term marks
  - Final exam marks
  - Assignments
  - Quizzes
  - Presentations
  - Practical (optional)
  - Project (optional)

- ⚖️ Custom Weightage System
  
  - Teachers can define weightage for each component
  - Fully flexible marking scheme

- 🧾 Student Records Management
  
  - Add student name and marks
  - Automatic calculations

- 📈 Automatic Calculations
  
  - Total marks
  - Sessional marks
  - Grade assignment
  - Class ranking

- 📄 PDF Export
  
  - Download results in PDF format
  - Easy sharing and record keeping

⚙️ How It Works (Workflow)

1. Teacher logs in or creates an account
2. Selects:
   - Department
   - Semester
   - Course details
3. Sets total marks and weightage for:
   - Mid, Final, Assignments, Quizzes, etc.
4. Enters student data (name + marks)
5. System automatically:
   - Calculates totals
   - Assigns grades
   - Generates class ranking
6. Teacher downloads results as a PDF



💡 Use Case

This system is useful for:

- University teachers
- College instructors
- Academic institutions
- Anyone managing sessional grading manually



🔧 Installation (Basic)

1. Clone the repository:
   
   git clone https://github.com/Abdur-Rehman-AR/Academic_Performance_Calculation_System

2. Setup XAMPP and start:
   
   - Apache
   - MySQL

3. Import database:
   
   - Open phpMyAdmin
   - Import ".sql" file

4. Run backend:
   
   npm install
npm start

5. Open in browser:
   
   http://localhost:3000/Home.html
